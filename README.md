# Secure Architecture & Threat Modeling
## University Management System (UMS)

This repository contains the full secure architecture design and formal threat modeling analysis for an internet facing University Management System.

The project includes:

- System architecture diagrams
- Trust boundaries
- Asset inventory
- STRIDE-based threat model
- Risk reasoning (Likelihood × Impact)
- Secure architectural controls
- Risk treatment decisions
- Microsoft Threat Modeling Tool model and exports

The system is cloud-agnostic and considers both external and insider threats.

See `report.md` for the full structured submission.
